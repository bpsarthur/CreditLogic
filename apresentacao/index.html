O que e o CreditLogic?

💰

Simulador de Credito

Um sistema que decide se uma pessoa tem o credito aprovado ou reprovado, assim como bancos e financeiras fazem na vida real.

🧮

Logica Proposicional

Utiliza 12 proposicoes logicas (verdadeiro ou falso) para tomar a decisao. E basicamente uma tabela verdade gigante aplicada na pratica.

📊

Painel Administrativo

Possui um dashboard completo com graficos, estatisticas e logs para o administrador acompanhar todas as analises realizadas.

Como funciona na pratica?

1

O usuario preenche um formulario com 12 perguntas sobre sua situacao financeira

➜
2

O sistema avalia as respostas usando a formula logica

➜
3

Resultado: APROVADO ou REPROVADO

Formula de Decisao

(P1 ∧ P2 ∧ P3 ∧ P6 ∧ P12) ∧ (total_verdadeiros ≥ 8)

As 5 proposicoes obrigatorias devem ser verdadeiras E pelo menos 8 das 12 no total.

Stack Tecnologica — Backend

O "cerebro" da aplicacao — o que roda por tras

Py

Python

A linguagem de programacao usada no servidor. E como o idioma que o computador entende pra executar as regras de credito.

Dj

Django

Um framework (conjunto de ferramentas prontas) que facilita criar o servidor web. Imagina como um kit de construcao LEGO pra montar o backend rapido.

API

Django REST Framework

Extensao do Django que cria as APIs — os "canais de comunicacao" entre o frontend e o backend. E como um garcom levando pedidos da mesa pra cozinha.

DB

SQLite

O banco de dados. E onde ficam salvas todas as analises de credito. Pense nele como uma planilha do Excel automatizada que guarda tudo.

Stack Tecnologica — Frontend

A "cara" da aplicacao — o que o usuario ve e clica

Re

React

Biblioteca do Facebook pra construir interfaces. E o que faz os botoes, formularios e paginas funcionarem. Imagina como os tijolos que montam a tela.

Vi

Vite

Ferramenta que empacota e serve o codigo React. E como um forno turbo — compila tudo muito rapido pra voce ver as mudancas na hora.

Ax

Axios

Biblioteca que faz as requisicoes HTTP. E o carteiro que leva os dados do formulario ate o servidor e traz a resposta de volta.

Ch

Recharts

Gera os graficos bonitos do painel admin. Transforma numeros chatos em graficos de pizza, barras e linhas visuais.

Por que essas tecnologias?

⚡ Django + DRF

Rapido de desenvolver, seguro por padrao, e o ecossistema Python e gigante.

⚙ React + Vite

Padrao da industria pra interfaces modernas. Componentes reutilizaveis e performance excelente.

📦 SQLite

Zero configuracao, perfeito pra projetos que nao precisam de um servidor de banco dedicado.

📈 Recharts

Graficos profissionais com poucas linhas de codigo. Integra nativamente com React.

💣

Tiro de Bazuca em Formiga

Vamos ser sinceros: usar Django REST Framework + React + Vite pra um projeto de faculdade e como usar um canhao pra matar um mosquito.

Essa stack e usada por empresas como Instagram, Netflix e Airbnb. Nos usamos pra avaliar 12 perguntas de sim ou nao.

Um simples formulario em HTML puro com um script de 50 linhas resolveria. Mas onde estaria a graca nisso? Fizemos por que podemos.
